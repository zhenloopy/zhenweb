tinyWeather

overview

tinyWeather gives weather notifications based on approximate location, daily and on startup using systemd timers.

technologies used

  • systemd services
  • python
  • curl and NWS, whereismyip API

Key Features

  • configurable frequency - documentation coming
  • uses notify-send to pop up notifs in your theme
  • customizable icon + messages
